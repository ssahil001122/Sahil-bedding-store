# Sahil Bedding Store

Professional e-commerce store for bedding products built with React.js

## 🚀 Features

- Modern, responsive design
- 8 product categories
- 16+ products with ratings & discounts
- Shopping cart & checkout
- Admin dashboard
- Flash sale section
- Trust badges
- Newsletter subscription

## 🛠️ Tech Stack

- React.js + Vite
- Tailwind CSS
- React Router
- React Icons

## 📦 Installation

```bash
npm install
npm run dev
```

## 🌐 Build

```bash
npm run build
npx serve dist
```
